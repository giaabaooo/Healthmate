# Screen Descriptions

Table III.8: Screen description

| # | Feature | Screen | Description |
|---|---|---|---|
| 1 | View Home Page | Landing Screen | Display the landing page and introduction to the HealthMate platform for guests. |
| 2 | Register Account | Register Screen | User registration interface to create a new system account. |
| 3 | Login | Login Screen | Authentication interface for users and admins to log into the system. |
| 4 | Complete Onboarding | Onboarding Screen | Interface for new users to fill in their initial physical metrics and health goals. |
| 5 | Update Profile | Profile Screen | Display user profile settings for viewing and updating personal information. |
| 6 | Set Fitness Goals | Fitness Goals Screen | Interface allowing users to create, view, and update fitness goals, AI roadmaps, and weekly check-ins. |
| 7 | Track Body Progress | Overview Screen | Dashboard displaying daily stats and allowing users to log weight, height, BMI, and body progress entries. |
| 8 | Create/Log Workout | Workout Manager Screen | Form interface to design new custom workouts and log completed workout sessions. |
| 9 | View Workouts List | Workouts Screen | Display lists of predefined system workouts and customized user routines. |
| 10 | View Workout Detail | Workout Detail Screen | Detailed view displaying information, calories, exercises, and instructions for a specific workout. |
| 11 | Search Food Catalog | Food Catalog Screen | Interface to browse the nutritional database and view calories and macronutrients of foods. |
| 12 | Create Meal Plan | Meal Planner Screen | Interface allowing users to build and modify daily meal plans for breakfast, lunch, dinner, and snacks. |
| 13 | View Analytics | Activity and Progress Dashboard Screen | Performance dashboard displaying charts and statistics on burned calories, workout activity, and body progress over time. |
| 14 | Manage Schedule | Schedule Screen | Display calendar view to plan and manage workout and meal schedules by week or month. |
| 15 | AI Assistance | AI Coach Screen | Chat and analytics interface allowing users to interact with the AI assistant for personalized health and fitness advice. |
| 16 | Access Community | Community Feed Screen | Display community posts and user progress, allowing members to share their own milestones and updates. |
| 17 | Manage Users | User Management Screen | Admin interface to search, view details, edit, disable, or manage user accounts. |
| 18 | Manage Food Catalog | Admin Food Catalog Screen | Admin interface displaying the global list of food items and their nutrition facts. |
| 19 | Add/Edit Food Item | Admin Food Form Screen | Admin form interface to create or edit detailed nutritional data for a specific food item. |
| 20 | Manage System Workouts | Admin Workouts Screen | Admin interface to add, edit, delete, and organize default workout templates for the system. |
| 21 | Manage User Meal Plans | Admin Meal Planner Screen | Interface allowing admins to oversee or assist in planning daily meals for selected users. |
| 22 | View Admin Dashboard | Admin Dashboard Screen | Dashboard interface displaying global system metrics, recent activities, and platform usage statistics. |

## Notes

- This table is prepared at screen level for report documentation.
- The wording has been aligned with the current HealthMate project structure and implementation.
- Some modules in the repository still contain prototype or legacy code, so this table prioritizes the main screens that are clearly represented in the current project.
